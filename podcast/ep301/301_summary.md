**Ep. 301 — The AI coding ladder**
*Fragmented · Kaushik & Yuri · 25 min · 2026-01-19*

Kaushik and Yuri lay out a four-generation framework for AI-assisted coding, arguing that most conversations about "AI coding" fail because people are talking about completely different paradigms.

**Gen 1** is autocomplete on steroids — GitHub Copilot-era token prediction trained on massive codebases. Unlike IntelliSense's rule-based, deterministic suggestions, AI autocomplete is probabilistic: the same comment could produce different completions each invocation. Cursor Tab is the modern refinement: a purpose-built fast model that lets you tab through cascading fixes in seconds. **Gen 2** is chat-oriented programming — the Cursor IDE model where a sidebar chat applies changes directly to your files. This kickstarted vibe coding by removing the copy-paste loop between ChatGPT and your editor, but kept you locked in a synchronous back-and-forth with the model. **Gen 3** is agentic coding: an LLM that reasons through a problem, creates a plan (often literally a markdown to-do file), then executes it in a loop using tools — think the ReAct (Reason + Act) framework. The verification loop is key: tests, linters, and type checkers give the agent immediate feedback so it can self-correct. Thinking models (DeepSeek, late 2024) made this viable by improving the planning step; you can literally watch the model hit an "aha moment" in its reasoning trace. Tools here split into TUI/CLI agents (Claude Code, Codex, Gemini CLI), IDE agents (Cursor, Junie, Windsurf), and headless agents (Google's Jules, which clones your repo and runs async).

**Gen 4** is still unsolved: multi-agent orchestration. The problem is coordination — multiple agents editing the same codebase produce merge conflicts just like multiple humans would. Git worktrees, stacked PRs, and agent-to-agent communication are all being explored, but no dominant pattern has emerged. The hosts frame this as the defining engineering challenge of 2026, compounded by the human context-switching ceiling of managing four or five parallel agent tasks simultaneously.

**Why it's worth your time:** If you want a clean mental model for placing any AI coding tool — or any colleague's workflow — on a progression from autocomplete to autonomous orchestration, this episode gives you the vocabulary in 25 minutes.
