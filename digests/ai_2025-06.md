### [Trailblaze: AI-Powered Mobile Testing with Natural Language](https://github.com/block/trailblaze)

Trailblaze (from Block) is an AI-powered UI testing framework that lets you write Android instrumentation tests in natural language instead of Espresso/UiAutomator API calls. Tests run on-device via standard instrumentation infrastructure, so they slot into existing CI pipelines and reporting systems without changes. The framework implements several techniques from Mobile-Agent-v3 research: automatic exception handling (popups, ads, loading states), reflection and self-correction when the agent detects a stuck or looping state, task decomposition for complex multi-step objectives, and cross-app memory for workflows that switch between apps. Under the hood it uses Maestro Command Models for UI interactions, which also provides a path toward cross-platform test reuse. Extend with custom `TrailblazeTool` implementations to give the agent app-specific knowledge. The gradual adoption path — starting with a handful of natural-language tests alongside existing Espresso tests — is the practical on-ramp.
