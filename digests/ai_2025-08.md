### [DevAnalyzer: Cross-Platform Developer Environment Analyzer](https://github.com/Coding-Meet/DevAnalyzer)

DevAnalyzer is a Compose Multiplatform / KMP desktop app that gives you a unified view of your Android development environment: project module structure, Gradle configurations, applied plugins, version catalogs, dependency trees, AGP/Kotlin/NDK/CMake versions, and SDK/IDE/Gradle cache storage consumption. The **Project Analyzer** inspects modules and build files; the **Clean Build** tool scans all modules for `build/` folders and lets you selectively delete them to reclaim disk space; the **Storage Analyzer** visualises total space used across SDKs, IDE data, Gradle caches, and library directories. Custom paths for Android SDK, Gradle User Home, Kotlin Native, and IDE locations are configurable. Built with Compose Multiplatform, so it runs on macOS, Windows, and Linux from the same codebase.
