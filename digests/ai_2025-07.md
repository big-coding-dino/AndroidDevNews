### [Llamatik: Kotlin Multiplatform LLM Inference Library](https://github.com/ferranpons/Llamatik)

Llamatik is a true KMP library that wraps `llama.cpp`, `whisper.cpp`, and `stable-diffusion.cpp` behind a unified Kotlin API, enabling fully on-device LLM inference, speech-to-text, and image generation across Android, iOS, Desktop, and WASM — no Python, no required server. Key capabilities: text generation (streaming and non-streaming), context-aware generation with system prompt and conversation history, schema-constrained JSON output, embeddings for vector search and RAG, on-device transcription from 16kHz mono WAV, and Stable Diffusion text-to-image. Models use GGUF format (LLMs) and BIN format (Whisper); integrated model download and management is included. An optional `llamatik-backend` HTTP server lets you swap between on-device and remote inference with the same API surface. JNI powers Android/Desktop; Kotlin/Native static frameworks power iOS. The main caveat: model files are large, so storage management and download strategy matter for production apps.
